# robomech-template

Latex template for Robomech

### 1. Edit LaTeX files

### 2. Make pdf

```bash
$ make
```

### Optional. Convert Japanese punctuations

```bash
$ make pub
# or
$ make publish
# will convert 「、」「。」 to 「，」「．」 in *.tex
# Original files are backed up as *.tex.orig
```

### Optional. cleaning

```bash
$ make clean
# or
$ make wipe
```
